# [ CSD ] Cit Software Distribution

**Um hub colaborativo de pesquisa e desenvolvimento de software.**

A CSD é mantida por um grupo de desenvolvedores independentes focados em expandir conhecimentos através de projetos práticos. Nós utilizamos este espaço para centralizar nosso portfólio conjunto, documentar nossos estudos e compartilhar código.

### >_ Foco Atual de Desenvolvimento

1.  **Segurança da Informação:** Hacking prático e análise de malwares.
2.  **Engenharia de Software:** Compiladores, parsers e gerenciamento de toolchains.
3.  **Computação Gráfica:** Renderização e matemática gráfica com OpenGL.
4.  **Sistemas em Produção:** Desenvolvimento de aplicações reais e otimização de infraestrutura.
